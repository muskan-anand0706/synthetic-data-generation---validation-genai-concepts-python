# 🤖 **Synthetic Data Generation & Validation using Generative AI Concepts** 📊  
---

# 📖 **Project Overview**🖊️
🧾This project demonstrates the application of **Generative AI concepts** for creating a synthetic dataset from a real-world HR dataset.  
🧾The objective is to generate **synthetic data** while maintaining the **statistical characteristics** of the original dataset.  

---

# 🎯 **Objectives**  
✅ Generate realistic synthetic data    
✅ Preserve statistical properties   
✅ Validate similarity using analytical techniques    

---
# 🛠 **Technologies Used**  
⏺️Python    
⏺️Pandas    
⏺️NumPy    
⏺️Matplotlib    
⏺️SciPy     


---    
# 📂 **Dataset**

The project uses an HR dataset containing:  

♦️Demographic features    
♦️Income-related attributes    
♦️Employee metrics    

---

# 🤖 **Synthetic Data Generation**

Synthetic data is generated by:

✔ Sampling numerical features using statistical distributions  
✔ Sampling categorical features based on probability distributions  
✔ Reconstructing dataset structure  

---  

# 📈 **Validation & Analysis** 📋

To evaluate realism, the following comparisons were performed:

📈 Mean Comparison  
📐 Variance Comparison  
🔥 Correlation Heatmap  
📊 Distribution Analysis  
🧪 Kolmogorov–Smirnov (KS) Test  

---  

# 🧪 **Statistical Similarity (KS Test)**  

🎀The KS Test was applied to measure distribution similarity between 
real and synthetic numerical features.  

🎀Higher p-values indicate statistical consistency between datasets.

---  

# 🪧**Key Outcomes**
🕹️Synthetic data resembles real-world patterns  
🕹️Statistical properties preserved    
🕹️Suitable for analytics & modelling tasks   

--- 

# 🏁 **Conclusion**

This project successfully demonstrates how **Generative AI concepts** 
can be combined with **data analytics validation techniques** to produce 
realistic synthetic datasets.
